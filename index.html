Linux Command Overview and Usage Guide

Basic File Commands

touch Command

Updates the timestamps of a file, useful for manually managing systems or rebuilding specific parts.

			
				touch name.txt        # Creates an empty file with the specified name
				touch a{1,7,9}.html   # Creates files named a1.html, a7.html, a9.html

echo Command

			
				echo {z..a}	# Outputs letters from 'z' to 'a' in reverse order
				echo a{A{1,2},B{3,4}}b    # Outputs nested patterns: aA1b aA2b aB3b aB4b

cp Command

Copies files.

			
				cp *.html Html       # Copies all .html files to the "Html" directory
				cp -i *.html html2   # Interactive copy (prompts before overwriting)

mv Command

Moves or renames files.

			
				mv *.html HTML-backup   # Moves all .html files to the "HTML-backup" directory

rm Command

Removes files and directories.

			
				rm a1.bak   # Removes the file a1.bak
				rm -rf /    # Caution! Deletes the entire filesystem

mkdir Command

Creates directories.

			
				mkdir {2019..2024}-{01..12} # Creates directories for each month from 2019 to 2024

Wildcards and Patterns

Using Wildcards

* matches zero or more characters, ? matches exactly one character.

			
				ls a*         # Lists all files starting with 'a'
				ls a?.html    # Lists files with 'a' followed by any one character and ending in .html

Utility Commands

ls Command

Lists directory contents.

			
				ls -a   # Shows all files, including hidden ones
				ls -l   # Long listing format

pwd Command

Displays the current working directory

			
				pwd
				/home/kali

cd Command

Changes the current directory.

			
				cd /          # Changes to the root directory
				cd ~          # Changes to the home directory

man Command

Shows the manual page for a command

			
				man du

Searching and Filtering

grep Command

Searches for patterns in files or outputs.

			
				history | grep ffuf    # Searches for "ffuf" in the command history
				history | grep -v ffuf # Searches for lines without "ffuf" in the command history

find Command

Finds files or directories based on specified criteria

			
				find . -type f -print                     # Finds files in the current directory
				find / -type f -name '<file>' 2>/dev/null # Finds a file from root (slow) and removes any errors

uniq Command

Filters out repeated lines

Expects sorted input

			
				sort -n nums.txt | uniq	# Removes duplicates after sorting by numeric

Redirection and Piping

Using Pipes and Redirection

Pipe

Sends the output of one command to another.

			
				cat file_list.txt | sort > sorted_file_list.txt # Sorts and redirects output

Redirection

Send input from a file or output to a file

			
				sort < file_list.txt > sorted_file_list.txt    # Sorts and redirects output

Disk and System Information

du Command

Displays disk usage

			
				du
				du | sort -nr   # Sorts by disk usage in descending order

wc Command

Counts words, lines, or characters in a file

			
				wc -l nums.txt   # Counts lines in nums.txt
				wc -w nums.txt   # Counts words in nums.txt
				wc -c nums.txt   # Counts characters in nums.txt

Can also be piped

			
				history | wc
				420    2118    17814
			      # lines  words   characters

Network and System Services

service Command

Manages services on Linux

			
				service apache2 start   # Starts the Apache2 service

updatedb Command

Updates the filesystem database

			
				updatedb

locate Command

Finds files by name using the updated database
find is more extensive than locate but much slower

			
				locate sbd

Web Scraping Example

Web Scraping with wget and grep

Download a webpage and extract specific links

				
					mkdir dom_scrape
					cd dom_scrape
					wget www.cisco.com
					grep 'href=' index.html | cut -d '/' -f3 | grep '\.cisco\.com' | cut -d '"' -f1 | sort -u > cisco_hosts.txt

Getting IPs from URLs

				
					for machine in  $(cat cisco_hosts.txt); do
							host $machine
					done	| grep 'has address' | cut -d ' ' -f4 | sort -u

Additional Commands and Shortcuts

type

Shows the type of a command.

				
					type ls
					ls is an alias for ls --color=auto

which

Locates the executable path of a command.

				
					which cp
					/usr/bin/cp

clear

Clears the screen

				 clear

history

Displays command history.

				 history
